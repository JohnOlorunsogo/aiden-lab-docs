# Training & Implementation Guide

This draft 4-module schedule is designed for Huawei ICT Academy-style delivery.

## 4-module course schedule

### Module 1: Lab Foundations and Topology Design

- Introduce AIDEN Lab architecture and deployment model.
- Build baseline enterprise topology in eNSP.
- Define addressing, VLAN, and routing plan.

### Module 2: Configuration and Service Bring-up

- Configure switching, routing, and policy controls.
- Validate traffic paths and fault baselines.
- Capture operational logs for AI analysis.

### Module 3: AI Troubleshooting Workflows

- Inject controlled failures (OSPF, VLAN, access policy errors).
- Use ENSP Logger insights to identify root cause.
- Compare manual diagnosis vs AI-assisted diagnosis.

### Module 4: Self-Healing Project and Evaluation

- Students design remediation playbooks.
- Build repeatable fix workflow with verification checkpoints.
- Present final self-healing project and operational report.

## Student workflow

1. Topology implementation.
2. Device/service configuration.
3. AI-assisted troubleshooting cycle.
4. Self-healing project execution and demonstration.

## Assessment checklist

- Correct topology and configuration standards.
- Incident response time and troubleshooting accuracy.
- Quality of remediation documentation.
- Ability to explain AI recommendations and limits.
