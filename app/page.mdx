## Architecture at a glance

```mermaid
flowchart LR
  subgraph onPrem [OnPremTrack]
    openEuler[openEulerServer]
    kubeVirt[PodmanAndKubeVirt]
    eNSPOnPrem[eNSPInstances]
  end
  subgraph cloudPath [CloudTrack]
    huaweiCloud[HuaweiCloudKubernetes]
    eNSPCloud[eNSPCloudNodes]
  end
  openEuler --> kubeVirt --> eNSPOnPrem
  huaweiCloud --> eNSPCloud
  eNSPOnPrem --> ai[AILoggerMindSporeLite]
  eNSPCloud --> ai
  ai --> gui[FlaskAndNoVNC]
```

Hybrid flow:

1. openEuler host provides a hardened base.
2. Podman and KubeVirt run isolated lab workloads.
3. eNSP instances host practical network scenarios.
4. AI Logger (MindSpore Lite on Ascend) inspects logs and flags faults.
5. Flask/noVNC web GUI exposes remote operation and troubleshooting.
